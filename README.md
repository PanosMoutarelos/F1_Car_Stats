# F1 Car Stats

A simple C# console application to track Formula 1 car statistics, calculate power-to-weight ratio, and compare different cars.

## Features
- Create F1 car objects with team name, horsepower, weight, year, constructor, and engine type.
- Calculate power-to-weight ratio (hp/kg).
- Compare power-to-weight ratio between two cars.
- Display a clean summary (stats report) of each car.

## How to Run
1. Open the project in Visual Studio.
2. Build the solution.
3. Run the program in the console.
4. Modify or add cars in `Program.cs` as needed.
