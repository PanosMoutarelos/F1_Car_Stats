# F1 Car Stats

A simple C# console application to track Formula 1 car statistics, calculate power-to-weight ratio, and compare two cars.

## Features
Input stats for two F1 cars via console:

-Team name
-Horsepower
-Weight
-Year of creation
-Constructor location
-Engine type
-Calculate power-to-weight ratio (hp/kg).
-Compare the power-to-weight ratio between the two cars.
-Display a clean summary of each car's stats.

## How to Run
1.Run the application.
2.Enter the stats for the first car when prompted.
3.Enter the stats for the second car when prompted.
4.The program will display:
5.Summary of both cars
6.Comparison of their power-to-weight ratio
